## :rocket: Getting Started

**Clone or Download the Repository**

```bash
$ git clone https://github.com/Johnny-Saraiva/trillo.git

$ cd trillo
```

**Installing Dependencies**

```bash
# if you use yarn, run this:

$ yarn
```
or
```bash
# if you use npm, run this:

$ npm install
```

### Run the project

```bash
# if you use yarn, run this:

$ yarn start

# if you use npm, run this:

$ npm run start

```
